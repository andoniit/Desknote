# App icons

Drop PNG icons here for PWA install + browser tabs.

Recommended set:

- `icon-192.png` — 192×192, maskable-safe, warm cream background
- `icon-512.png` — 512×512, maskable-safe
- `favicon.ico` — 32×32 / 48×48 multi-size

You can generate them from a single SVG logo at a tool like
https://realfavicongenerator.net/.
